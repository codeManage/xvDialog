#xvDialog.js一款基于jquery轻量级的弹层插件
<br>
***
<br>
xvDialog.js一款基于jquery轻量级的弹层插件,需要引入插件 xvDialog.js/xvDialog.min.js,需要引入 xvDialog.css/ xvDialog.min.css 样式表，可以自定义自己想要的皮肤 ( 本弹出层组件基于jquery库开发版本1.8.0以上 )
<br>
